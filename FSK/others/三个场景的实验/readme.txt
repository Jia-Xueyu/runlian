1. matlab的各个文件负责生成转化音频，播放音频，接收音频，解码等各种任务。
2. python的各个文件负责并行处理任务和判断接收任务。
3. notation.m下的
if M-X>1000*X,
    flag=1;
end;
这段代码是阈值判断，是判断解码后这段波形为1还是0。因为接与收设备是分开的，所以其实声音传输的误差还是蛮大的。而且不同电脑的扬声器
质量不同，发出的声音未必是想要频段的声音。反正声音设备传输的误差很大，受环境和设备的影响也很大，经常这个阈值要手动更改。更改方法
是可以把波形打印出来，然后自己来判断。影响因素蛮多，确实很难跑完整一次，这是我自己写的代码，完整跑通也只是成功了五次左右。
4. 传输过程是每秒传输一个比特，解码过程中把单位时间内的所有波的值平均来作为这段时间的波的峰值。解码方法你们可以选择其他的。
5. 最初我拿到的源代码是FSK.m，你们也可以根据这个原代码来完成自己的代码内容。
6. 选择的是18-19khZ和20-21khZ两个传输频段。
7.一个设备运行transmitter.py,一个设备运行receiver.py,一个设备运行disturber.py。运行完后会出现information.txt和information_original.txt，再运行BER.m
可以得到误码率。
